1. Load + inspect

Goal:

load CSV
print shape
print column names
show first 5 rows


2. Column exploration

Goal:

select 2–3 columns
print summary of each
check data types


3. Missing values audit

Goal:

count missing values per column
identify top 3 columns with most missing data


4. Basic statistics

Goal:

compute mean, median, std for numeric column(s)



5. Feature creation

Goal:

create new column from existing ones
(e.g. age categories, ratios, flags)



6. Filtering rows

Goal:

filter dataset using at least 2 conditions
(e.g. Age > 30 AND survived == 1)



7. Groupby aggregation

Goal:

group by categorical column
compute mean of numeric column per group


8. Sorting + ranking

Goal:

sort dataset by numeric column
extract top 10 rows
extract bottom 10 rows




9. Simple visualization

Goal:

create at least 2 plots:
histogram
bar chart or scatter plot


10. Mini analysis conclusion

Goal:

write 3–5 lines of findings:
what stands out in data
one correlation or pattern
one potential insight
